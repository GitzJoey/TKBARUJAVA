kitUtils 1.2.0

Device checking

kitUtils.isAndroid();

kitUtils.isIOS();

kitUtils.isBlackBerry();

kitUtils.isIEMobile();

kitUtils.isOperaMini();

kitUtils.isMobileBrowser();

Internet Explorer Version

kitUtils.detectIEVersion(); Adds the Internet Explorer version as a class on the document element.

EPiServer Iframe Check

kitUtils.episerverIframeCheck(); Adds .episerver on the body element.

Last Child Fix

kitUtils.lastChildFix(); Adds .last-child on all elements with :last-child. A fix for IE8.

Safe Logging

kitUtils.log('My console message'); Logs messages and objects in browsers with console support.

You can also provide two more parameters, data and type. Both are optional

kitUtils.log('My console message', {foo: 'bar'}, 'info');
